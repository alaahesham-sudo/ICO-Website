# In-Call Outsourcing LLC - Healthcare BPO Services

Professional healthcare BPO and telemarketing services website built with Next.js and Tailwind CSS.

## Features

- Modern, responsive design
- Healthcare BPO service information
- Careers application form
- Contact form with Google Sheets integration
- Team gallery and testimonials
- Dynamic statistics visualization

## Tech Stack

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- Google Apps Script (form submissions)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deployment

Deployed on Netlify with automatic deployments from GitHub.
